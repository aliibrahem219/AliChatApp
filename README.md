# AliChatApp
A MERN stack chatapp using socket.io and another powerfull express libraries like bcryptjs,cloudinary,cookie-parser,cors,dotenv,jsonwebtoken and nodemailer and react librares like socket.io-client,react-router-dom and simple-peer for implementation of a WebRTC video call
The app needs to signup before start using username,email,password and a bio after that you can starting chat with other users and you can verify your email and reset your forgot password and update your profile photo,username and bio
